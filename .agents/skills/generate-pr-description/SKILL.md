# Skill: Generate PR Description

Use this skill after implementation, review, tests and lint.

Read:

- Jira issue
- `.agents/workflow/current.md`
- PRD
- ADRs
- Design Doc
- git diff
- test results
- lint results

Generate:

1. Commit message
2. PR title
3. PR description

PR description must include:

- Jira ID
- Summary
- Affected projects
- Docs
- Tests
- Lint/typecheck
- Risks
- Rollback notes
