# Skill: Create ADR

Create an ADR only when there is a long-term technical decision.

Find next ADR number:

1. List `docs/adrs/`.
2. Find the highest numeric prefix.
3. Use the next number.
4. Format with four digits: `0001`, `0002`, `0003`.

Path:

`docs/adrs/{ADR_NUMBER}-{JIRA_ID optional}-{decision-slug}.md`

Template:
